# Version 2 Real Model Demo

This package demonstrates real issue-to-plan generation through Hugging Face Inference Providers.

- `issue.md`: issue and supplied repository context.
- `plan.md`: unedited model response.
- `mock_baseline.md`: Sprint 1 fallback output for the same issue.
- `comparison.md`: concise qualitative comparison for the report.
- `metadata.json`: model, provider, latency, token usage, and validation result.

The response was generated by `Qwen/Qwen3-Coder-30B-A3B-Instruct` through Scaleway using `context_AI/ml/autogen_prompt.md`.
